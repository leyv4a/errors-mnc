Project Details
Aplicación de Errores

Este proyecto está diseñado para calcular y mostrar errores absolutos, relativos y porcentuales. Permite ingresar dos números (valor calculado y valor real), calcular los errores y mantener un historial de cálculos.
Stack Utilizado

    Next.js: Framework React
    Shadcn: Componentes UI
    Tailwind.css: Estilos utilitarios
    TypeScript: Tipado estático
    Vercel: Hospedaje web
    Lucide: Iconos UI
    ESLint: Calidad código

Instrucciones

    Recibir 2 números: Valor calculado y valor real con hasta 4 decimales.
    Calcular errores: Error absoluto, error relativo y error porcentual.
    Guardar historial: Se guarda un historial de valores y errores.
    Ver historial: Un botón para ver el historial.
    Limpiar historial: El historial se limpia cuando se cierra la aplicación.
